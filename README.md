### Zależności
- requests
- beautifulsoup4
- pandas
- lxml

### Jak używać?

### Sposób 1
- odpal cmd w katalogu skryptu
- zainstaluj zależności:
pip install requests beautifulsoup4 lxml pandas
- po zainstalowaniu zależności wpisz:
py tge_scraper.py

### JESLI POPRZEDNI SPOSOB NIE DZIALA ZROB TO:
- odpal cmd w katalogu skryptu
- stworz virtual environment (dla pythona jest to venv)
- dla windowsa:
python -m venv venv
venv\Scripts\activate
- dla linuxa:
python -m venv venv
source venv/bin/activate
- nastepnie w venv zainstaluj zależności:
pip install requests beautifulsoup4 lxml pandas
- po instalacji wszystko powinno działać i wystarczy włączyć skrypt komendą:
py tge_scraper.py
